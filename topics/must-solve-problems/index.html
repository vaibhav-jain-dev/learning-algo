Topic Categories

Linear Data Structures

Arrays - Two pointers, sliding window, prefix sums, in-place modifications (40-50% of all interview questions) Linked Lists - Fast/slow pointers, reversal, cycle detection, merge operations

Tree Structures

Binary Trees - Traversals (inorder, preorder, postorder, level-order), path problems, tree construction Binary Search Trees - BST properties, validation, LCA, iterators, balancing

Graph Algorithms

Graphs - BFS, DFS, topological sort, shortest paths, union-find, cycle detection

Algorithmic Paradigms

Recursion & Backtracking - Subsets, permutations, combinations, N-Queens, Sudoku solver Dynamic Programming - Memoization, tabulation, state transitions, optimization problems Famous Algorithms - Classic algorithms every programmer should know (Dijkstra, Kadane, KMP, etc.)

Pattern Recognition Framework

PATTERN IDENTIFICATION FLOWCHART

If you see...

  • Sorted array + target = Binary Search
  • Pairs/triplets = Two Pointers
  • Contiguous subarray = Sliding Window
  • Optimal substructure = DP
  • Find all combinations = Backtracking

If problem asks for...

  • Shortest path = BFS
  • All paths = DFS/Backtracking
  • Top K elements = Heap
  • Common elements = Hash Set
  • Dependencies = Topological Sort
