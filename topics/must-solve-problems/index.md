# Must Solve Problems

## Overview

This curated collection of essential Data Structures and Algorithms (DSA) problems covers the patterns and techniques most frequently tested in technical interviews. Problems are organized by topic and progress from easy to very hard, building intuition for pattern recognition and problem-solving strategies.

|-------|-------------|-------------|----------------|
| **Easy** | Single technique, direct application | 15-20 min | Two Sum, Reverse List, Tree Traversal |
| **Medium** | Combine 2-3 techniques | 25-35 min | 3Sum, LRU Cache, Word Search |

<h4 style="color: #16a34a; margin: 0 0 12px 0;">If you see...</h4>
<ul style="color: #475569; font-size: 13px; margin: 0; padding-left: 20px; line-height: 2;">

<li>All paths = DFS/Backtracking</li>
<li>Top K elements = Heap</li>

## Content Format

Each topic includes:

- **Pattern Templates**: Reusable code patterns with explanations
- **Problem Lists**: Organized by difficulty with LeetCode links
- **Visual Explanations**: ASCII diagrams and step-by-step walkthroughs
- **Complexity Analysis**: Time and space breakdown
- **Interview Tips**: Common pitfalls and optimization hints
