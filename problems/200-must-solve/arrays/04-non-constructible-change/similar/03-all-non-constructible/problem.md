# All Non-Constructible Values

**Difficulty:** Hard

## Problem Statement

Find all values up to a given limit that cannot be constructed from the given coins.

## Examples

**Example 1:**
```
Input: coins = [1, 2, 5], limit = 10
Output: [] (all values 1-10 can be constructed)
```

**Example 2:**
```
Input: coins = [1, 5, 10], limit = 20
Output: [7, 8, 9, 17, 18, 19, 20]
```

## Constraints

- 0 <= coins.length <= 10^5
- 1 <= coins[i] <= 10^9
- 1 <= limit <= 10^6
