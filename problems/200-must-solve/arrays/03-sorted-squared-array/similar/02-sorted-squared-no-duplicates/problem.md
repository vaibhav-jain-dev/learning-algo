# Sorted Squared Array Without Duplicates

**Difficulty:** Medium (Yellow)

## Problem Statement

Given a sorted array of integers, square all elements and return a sorted array with duplicates removed.

## Examples

**Example 1:**
```
Input: array = [-3, -2, -1, 1, 2, 3]
Output: [1, 4, 9]
Explanation: -1 and 1 both give 1, -2 and 2 both give 4, etc.
```

**Example 2:**
```
Input: array = [-5, -3, 0, 2, 3, 5]
Output: [0, 4, 9, 25]
```

## Constraints

- Input array is sorted in ascending order
- Can contain negative integers and zero
